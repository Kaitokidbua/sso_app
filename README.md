# 🏥 SSO Smart Planner (ประกันสังคมฉบับประชาชน)

โปรเจกต์วิเคราะห์ข้อมูลและวางแผนสวัสดิการประกันสังคมไทย เพื่อช่วยให้ผู้ประกันตนเข้าถึงสิทธิประโยชน์ได้ง่ายขึ้น

## ✨ ฟีเจอร์หลัก
1. **Dashboard วิเคราะห์เทรนด์:** ดูการเคลื่อนย้ายแรงงานรายจังหวัดย้อนหลัง 3 ปี (2565-2567)
2. **AI Advisor:** ปรึกษาปัญหาเกี่ยวกับประกันสังคมผ่านแชทบอทที่เข้าใจบริบทข้อมูลจริง
3. **Welfare Planner:** เครื่องมือคำนวณเงินบำนาญชราภาพแบบเข้าใจง่าย

## 🛠 เทคโนโลยีที่ใช้
- **Frontend:** Streamlit (Theme: Dark Mode)
- **Database:** DuckDB (ใช้ Query ไฟล์ Excel ความเร็วสูง), MongoDB, Snowflake และ SQLite
- **Data:** Pandas & Plotly
- **AI:** Google Gemini API (หรือ OpenAI)

## 🚀 วิธีติดตั้ง
1. Clone repository: `https://github.com/Kaitokidbua/sso-project`
2. ติดตั้ง library: `pip install -r requirements.txt`
3. รันแอป: `streamlit run app.py`

## 📊 ข้อมูลที่ใช้อ้างอิง
- SSO Data Catalog (data.go.th) ปี 2565-2567
- เว็บไซต์สำนักงานประกันสังคม https://www.sso.go.th/wpr/

# SSO Smart Planner (ประกันสังคมฉบับประชาชน)

## Issues and motivation
1. ผู้ประกันตนจำนวนมากยังไม่สามารถประเมินความพร้อมทางการเงินหลังเกษียณได้อย่างชัดเจน เนื่องจากระบบประกันสังคมปัจจุบันมุ่งเน้นการให้ข้อมูลสิทธิประโยชน์ แต่ยังขาดเครื่องมือสำหรับคำนวณและคาดการณ์เงินบำนาญในอนาคต จึงมีความจำเป็นในการพัฒนาระบบที่ช่วยสนับสนุนการวางแผนเกษียณและการตัดสินใจทางการเงินระยะยาว
2. สิทธิประโยชน์ของผู้ประกันตนตามมาตรา 33, 39 และ 40 มีเงื่อนไขและความคุ้มครองที่แตกต่างกัน ส่งผลให้ผู้ใช้งานจำนวนมากเกิดความสับสนในการทำความเข้าใจและเปรียบเทียบสิทธิของแต่ละมาตรา จึงควรมีเครื่องมือที่สามารถสรุปและเปรียบเทียบข้อมูลในรูปแบบที่เข้าใจง่าย
3. การเข้าถึงข้อมูลประกันสังคมยังต้องอาศัยการค้นหาจากเอกสารหรือหลายแหล่งข้อมูล ทำให้ผู้ใช้งานใช้เวลามากและอาจได้รับข้อมูลไม่ครบถ้วน ดังนั้นการนำเทคโนโลยี AI มาช่วยตอบคำถามและให้คำแนะนำจึงสามารถเพิ่มความสะดวกและประสิทธิภาพในการเข้าถึงข้อมูลได้

## Objective
1. พัฒนาเว็บแอปพลิเคชันสำหรับรวบรวมและนำเสนอข้อมูลประกันสังคมในรูปแบบที่ใช้งานง่ายและเข้าถึงได้สะดวก
2. พัฒนาระบบคำนวณและคาดการณ์เงินบำนาญหรือผลประโยชน์ชราภาพจากข้อมูลของผู้ประกันตน เพื่อช่วยในการวางแผนเกษียณ
3. สร้าง Interactive Dashboard สำหรับสรุป วิเคราะห์ และเปรียบเทียบสิทธิประโยชน์ของผู้ประกันตนตามมาตรา 33, 39 และ 40 
4. พัฒนาระบบ AI Chatbot ที่สามารถตอบคำถามเกี่ยวกับประกันสังคม และให้คำแนะนำเบื้องต้นแก่ผู้ใช้งาน

## Solution (Methodology)
1. Non AI
สร้างระบบ Dashboard สำหรับวิเคราะห์และแสดงผลข้อมูลประกันสังคม โดยใช้ฐานข้อมูล SQL และ Snowflake ในการจัดเก็บและประมวลผลข้อมูล ผู้ใช้งานสามารถดูข้อมูลภาพรวมของผู้ประกันตน เช่น จำนวนผู้ประกันตนในแต่ละมาตรา (33, 39 และ 40) สัดส่วนการใช้สิทธิประโยชน์ การกระจายตัวตามจังหวัด และสถิติที่เกี่ยวข้อง นอกจากนี้ยังมีฟีเจอร์เปรียบเทียบสิทธิประโยชน์ระหว่างมาตราต่าง ๆ รวมถึงเครื่องมือช่วยวางแผนการเกษียณอายุ เพื่อสนับสนุนการตัดสินใจและการวางแผนทางการเงินของผู้ประกันตน
   
2. AI
พัฒนา AI Chatbot โดยใช้ Large Language Model (LLM) ผ่าน Gemini API เพื่อให้ผู้ใช้งานสามารถสอบถามข้อมูลเกี่ยวกับประกันสังคม เช่น สิทธิประโยชน์ เงื่อนไขการรับสิทธิ และรายละเอียดของผู้ประกันตนตามมาตรา 33, 39 และ 40 ได้อย่างสะดวกและรวดเร็ว พร้อมทั้งจัดเก็บประวัติการสนทนาและคำถามของผู้ใช้งานใน MongoDB เพื่อนำมาวิเคราะห์คำถามที่พบบ่อย (FAQ) สำหรับอำนวยความสะดวกกับผู้ใช้คนอื่น ๆ
